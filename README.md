# 防災ライブラリプロジェクト (blib)

## 🎯 プロジェクトの目的
**blib** は、日本全国の防災計画、法令、ガイドライン、および事業継続計画（BCP）を網羅的に集約・整理したデジタルライブラリです。
単なる資料の蓄積に留まらず、AIアシスタントとの共同作業や自動更新ワークフローを通じて、常に最新かつ信頼性の高い防災知識ベースを構築することを目指しています。

特に**東京都および小平市**の資料を最優先にカバーし、有事の際の迅速な情報参照を支援します。

## 📂 ライブラリの構成
資料は以下の8つのカテゴリに基づいて体系的に分類されています。

1. **1_全体枠組みと基本方針**: 災害対策基本法、国土強靱化基本法など
2. **2_災害直後の応急対応・避難**: 避難情報ガイドライン、受援体制、避難所運営など
3. **3_被災者支援と生活再建**: 災害救助法、被災者生活再建支援法、罹災証明など
4. **4_災害種別ごとの特化法・指針**: 火山対策、広域降灰、水防法、土砂災害防止法など
5. **5_建築物・まちづくりの安全確保**: 建築基準法、耐震改修促進法、密集市街地対策など
6. **6_多様性・コミュニティへの配慮**: 福祉避難所、男女共同参画、要配慮者支援など
7. **7_我が国の防災計画体系**: 防災基本計画、各省庁業務計画、地方自治体の地域防災計画
8. **8_事業継続計画（BCP）**: 自治体BCP、および民間企業のBCP事例

## 🛠 主な機能・特徴
- **原本保持（原本主義）**: 外部リンクに頼らず、PDF等の資料をローカルに保持することでオフラインでの参照や将来の消失リスクに対応しています。
- **Web UI インデックス**: `index.html` を通じて、ブラウザから直感的に資料を検索・閲覧可能です。
- **更新ウォッチリスト**: `UPDATE_WATCHLIST.md` により、資料の最新性を継続的に管理しています。
- **プレミアム・デザイン**: ダークネイビーを基調とした、視認性の高いモダンなインターフェースを採用しています。

## ⚠️ 注意事項（重要）
本リポジトリに格納されている各種資料（PDF等）は、中央省庁や地方自治体等の**公式サイトからダウンロードしてきた複製**です。

- 格納されている情報の正確性、最新性、および完全性を保証するものではありません。
- **法的判断や公的な手続き、あるいは人命に関わる重要な意思決定を行う際は、必ず発行元（内閣府、各省庁、自治体等）が公開している正式な一次情報を参照してください。**
- 各資料の著作権および利用規約は、それぞれの発行元に帰属します。

## 🚀 使い方
1. リポジトリをクローンまたはダウンロードします。
2. `index.html` をブラウザで開くと、整理されたライブラリのインデックスが表示されます。
3. カテゴリを選択、または検索バーを使用して必要な資料を探してください。
